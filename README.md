Анализ рейтингов брендов

    Данный проект, написанный на языке программирования Python, анализирует рейтинги брендов из CSV-файлов.

Установка

    Клонируйте репозиторий: git clone <url репозитория>
    Установите зависимости: pip install -r requirements.txt

Использование

    Нужно запустить анализ через терминал в PyCharm (или командную строку):

    python (или python3) -m src.main --files data/products1.csv data/products2.csv --report average-rating

Тестирование

    Для запуска тестов нужно ввести в терминале в PyCharm (или командную строку) команду pytest.
